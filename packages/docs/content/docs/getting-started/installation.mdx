## Prerequisites

- Node.js >= 20
- pnpm 9.x (`corepack enable` to use the pinned version)
- Docker and Docker Compose (for PostgreSQL, Redis, Keycloak, MinIO)

## Setup

Clone the repository and install dependencies:

```bash
git clone <repo-url>
cd Vastu
pnpm install
```

## Start Docker services

The project requires PostgreSQL, Redis, Keycloak, and MinIO. Start them with Docker Compose:

```bash
docker compose up -d
```

Services will be available at:

| Service | URL | Notes |
|---------|-----|-------|
| PostgreSQL | `localhost:5432` | Database |
| Redis | `localhost:6379` | Session/cache store |
| Keycloak | `localhost:8080` | Auth provider (realm: `vastu`) |
| MinIO | `localhost:9001` | Object storage (bucket: `vastu-uploads`) |

## Database setup

Generate the Prisma client and run migrations:

```bash
pnpm prisma:generate
pnpm prisma:migrate
pnpm prisma:seed
```

## Start development server

```bash
pnpm dev
```

The app will be available at [http://localhost:3000](http://localhost:3000).

## Common commands

```bash
pnpm build            # Build all packages
pnpm test             # Run all unit tests (Vitest)
pnpm test:e2e         # Run E2E tests (Playwright)
pnpm lint             # ESLint + Prettier check
pnpm lint --fix       # Auto-fix lint issues
pnpm typecheck        # TypeScript strict check
```
